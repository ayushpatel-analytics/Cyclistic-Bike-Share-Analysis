# Cyclistic Bike-Share: Comparative Behavioral Analysis

### Project Overview
This analysis explores the behavioral differences between *Annual Members* and *Casual Riders* to support a conversion-based marketing strategy. By analyzing over 5 million rows of trip data, this project identifies "conversion triggers" to transition casual users into long-term subscribers.

### Key Findings
1. *Usage Patterns:* Members show consistent "commuter" peaks (8 AM / 5 PM), while Casual riders peak during weekend afternoons.
2. *Trip Duration:* Casual riders maintain significantly higher average trip durations compared to Annual Members.
3. *Geographic Hotspots:* Casual usage is concentrated near leisure and tourist stations, whereas Members cluster around transit hubs.
