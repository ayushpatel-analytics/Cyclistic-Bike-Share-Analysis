# Cyclistic Bike-Share: Comparative Behavioral Analysis

### Project Overview
This analysis explores the behavioral differences between *Annual Members* and *Casual Riders* to support a conversion-based marketing strategy[span_1](start_span)[span_1](end_span)[span_2](start_span)[span_2](end_span). By analyzing over 5 million rows of trip data, this project identifies "conversion triggers" to transition casual users into long-term subscribers[span_3](start_span)[span_3](end_span)[span_4](start_span)[span_4](end_span).

### Key Findings
1. *Usage Patterns:* Members show consistent "commuter" peaks (8 AM / 5 PM), while Casual riders peak during weekend afternoons[span_14](start_span)[span_14](end_span).
2. *Trip Duration:* Casual riders maintain significantly higher average trip durations compared to Annual Members[span_15](start_span)[span_15](end_span).
3. *Geographic Hotspots:* Casual usage is concentrated near leisure and tourist stations, whereas Members cluster around transit hubs[span_16](start_span)[span_16](end_span).
